    Reference data

    The reference data is used to define some of the criteria thresholds. The identifiers in the reference data column of the thresholds correspond to the dataset identifiers listed below.

    Dataset Source Description
    CEDS-2025 Hoesly (2025) Based on Hoesly (2025).
    CEDS-2025-COVIDaveraged Hoesly (2025) Data from Hoesly (2025) averaged over period Jan 2018 till Dec 2022 (to account for the idiosyncratic COVID-19 dip in 2020).
    CEDS-2025-extrapol25 Hoesly (2025) Data from Hoesly (2025) linearly extrapolated to 2025.
    IEA-EB-2025 IEA (2025) Based on IEA (2025).
    IEA-EB-2025-COVIDaveraged IEA (2025) Data from IEA (2025) averaged over period Jan 2018 till Dec 2022 (to account for the idiosyncratic COVID-19 dip in 2020).
    IEA-EB-2025-extrapol25 IEA (2025) Data from IEA (2025) extrapolated to 2025 using the 2021-2023 average growth rate.
